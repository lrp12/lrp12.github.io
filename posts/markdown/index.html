Markdown 速查表提供了所有 Markdown 语法元素的基本解释
May 12, 2024 · RP

总览 Markdown 速查表提供了所有 Markdown 语法元素的基本解释。如果你想了解某些语法元素的更多信息，请参阅更详细的 基本语法 和 扩展语法.

基本语法#

这些是 John Gruber 的原始设计文档中列出的元素。所有 Markdown 应用程序都支持这些元素。

元素 Markdown 语法
标题（Heading） # H1## H2### H3
粗体（Bold） bold text
斜体（Italic） italicized text
引用块（Blockquote） > blockquote
有序列表（Ordered List） 1. First item 2. Second item3. Third item
无序列表（Unordered List） - First item- Second item- Third item
代码（Code） code
分隔线（Horizontal Rule） —
链接（Link） title
图片（Image） ![alt text](image.jpg)
